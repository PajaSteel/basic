LESSON 01 — LET

LET uloží hodnotu do proměnné.

10 LET A = 5
20 LET A = A + 3
30 PRINT A

Výsledek: 8
10 LET A = 7
20 LET A = A + 5
30 LET A = A * 2
40 PRINT A

LESSON 02 — IF ... THEN

IF ověří podmínku. Pokud platí, provede se část za THEN.

10 LET X = 4
20 LET B = 2
30 IF X < 10 THEN LET B = B + 11
40 PRINT B

LESSON 03 — FOR ... NEXT

FOR ... NEXT opakuje blok příkazů.

10 LET C = 2
20 FOR I = 1 TO 4
30 LET C = C + 3
40 NEXT I
50 PRINT C

LESSON 04 — GOTO

GOTO skočí na zadané číslo řádku.

10 LET D = 5
20 GOTO 50
30 LET D = D * 9
40 PRINT D
50 LET D = D + 8
60 PRINT D

LESSON 05 — DATA / READ

DATA obsahuje hodnoty, READ je načítá.

10 DATA 8,3,6
20 READ X,Y,Z
30 LET E = X * Y - Z
40 PRINT E

LESSON 06 — GOSUB / RETURN

GOSUB skočí do podprogramu a RETURN se vrátí zpět.

10 LET F = 4
20 GOSUB 100
30 LET F = F + 1
40 PRINT F
50 END

100 LET F = F * 3
110 RETURN
